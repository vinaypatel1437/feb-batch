
    1. querySelector :- Wo element utha ke lata hai jo usko pehle milta hai
    2. querySelectorAll :- Sare element uthata hai or ek array ke form me lake dedeta hai.
    3. getElementById :- Id match krta hai or uske corresponding jo bhi cheez lake dedega.
    4. addEventListener :- Hum jo bhi event lagate hai usko listen krta hai.
    5. innerHTML :- Tag ke andar se hame HTML ka code lake dedega.
    6. innerText :- Tag ke andar se hame Text lake dedega.
    7. getElementByClassName :- Class match karke querySelectorAll jese array lake dedega.
    8. value :- input tag ki value ko lake dedega .
    9. event :- event jo hai wo sirf or sirf form tag me work krta hai.
    10. Events:- click,onmouseenter,mouseleave.
    11. let,var,const,kuch bhi nahi.
